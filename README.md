# 🏎️ F1 Website

A responsive multi-page Formula 1 themed website built using HTML5, CSS3, Bootstrap 5, and Responsive Web Design principles.

---

## 📌 Project Overview

This project was created for a Web Development Terminal Assessment.

The website includes:

- Responsive Design
- Multi-page Navigation
- Bootstrap Components
- Mobile-First Layout
- Functional Contact Form UI
- Custom Styling

---

## 📄 Pages Included

### 🏠 Home Page
- Hero section
- Navigation bar
- Feature cards
- Responsive layout

### ℹ️ About Page
- Information about Formula 1
- Responsive image section
- Two-column layout

### 📩 Contact Page
- Contact form
- Input fields
- Submit button
- Mobile responsive form design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Flexbox
- Responsive Web Design

---

## 📱 Responsive Features

The website is fully responsive and works on:

- Mobile Phones
- Tablets
- Laptops
- Desktop Screens

Media queries were used to improve layouts on smaller screens.
